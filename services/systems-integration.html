FAQ

Common Questions

Can you integrate with legacy or proprietary systems?

Yes. As long as the system has some form of accessible interface — API, database access, file export, or even screen scraping as a last resort — we can build a connector for it.

What happens when an integration breaks?

We monitor all integrations 24/7 with automated error detection. When something fails, we receive an alert and begin investigating before you even notice. We also build in automatic retry logic and data recovery for most failure scenarios.

Is my data safe during integration?

All data transfer uses encrypted, authenticated connections. We never store your data on shared infrastructure, and we use the minimum permissions required for each integration to function.
